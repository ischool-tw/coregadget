gadget-學生排調代課小工具
==========================

* 身份「學生」
* 運用 javascript 語法來撰寫 javascript
* 運用 css 語法來撰寫 css
* 運用 gadget 物件呼叫 dsa 服務，依據呼叫不同類型的服務，來操作"查詢"等功能

功能說明
-------
**學期課表**
呈現目前學年度學期的課表

**查詢課表**
請輸入您要查詢的教師、班級、或場地的名稱後，指定單一「教師」、「班級」、或「場地」後，呈現目前學年度學期的課表

**調代課歷程**
呈現目前學年度學期課表中的調課、代課記錄

**調代課通知**
呈現「本週」及「未發生」的修課所有調代課通知
