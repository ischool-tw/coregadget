功能說明
-------

> 為新民高中客製小工具，每二個星期填寫一次，一個班級會有一位團膳股長，在班上進行投票後，負責填寫至系統。

 - 作答學生為班級的團膳股長
 - 每份問卷僅能作答一次，送出前需檢查必填是否完整填寫
 - 回傳時紀錄時間
 - 問卷開放期間內，可以重複編輯。目前預設一次只會開放一張問卷
 - 廠商為表單內容之一，與問卷無關；廠商可由Desktop設定隱藏
 - 目前題型有「單選、問答」，單選為必填，問答題可由Desktop設定為必填或非必填
