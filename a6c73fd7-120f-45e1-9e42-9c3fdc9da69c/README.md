gadget-重補修缺曠查詢
==========================

* 身份「學生、家長」
* 運用 javascript 語法來撰寫 javascript
* 運用 less css 語法來撰寫 css
* 運用 gadget 物件呼叫 dsa 服務，取得學生資訊
* 運用 twitter bootstrap 做為畫面設計的板型


----------


功能說明
-------

**gadget.paramValues 設定**

    //使用小工具者的身份
    "system_position": ["student"(學生，預設) || "parent"(家長)]

**重補修缺曠查詢**

 * 依梯次 > 課程名稱 呈現缺曠狀態
 * 可瀏覽全部課程，以日期遞減排序
 * 可瀏覽單一課程，以日期遞減排序
